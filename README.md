# Face Swap 流程

## 用户的基本使用流程

1. 用户拍摄一张正脸照片 (只执行一次)
2. 用户选择一个只有一个人视频模板
3. 用自己的脸替换视频中的人脸, 生成新的视频
4. megviifacepp_0_5_2_model 网络下载

## api接口

1. 人脸拍照之后将图片提交给face++ : recognizeFace(byte[] nv21Data)
2. 返回nv21格式的视频帧数据: byte[] decodeFrame() 
3. 编码opengl 替换之后的视频帧
4. 生成新的视频文件


## 后期优化 ( 第一期可能完成不了 )

1. 用户选择模板之后就可以看到脸部替换的效果, 手动解码-人脸识别-替换人脸-展示
2. 点击确认时生成新的视频
3. 后端为每一个模板生成一个人脸关键点序列, 避免客户端实时监测人脸, 减少消耗流量、降低延迟
